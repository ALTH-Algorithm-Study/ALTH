## 😎 운영 방식

1. 일주일에 한 챕터씩, 그 챕터에 해당하는 문제를 푼다. 
2. 스터디원은 하루에 하나 이상 씩 문제를 가져와서 백준 그룹에 문제를 게시한다. (전주 일요일에 시간제한 없이 올리도록 한다.)
3. 줌을 이용한 모임 시, 이전 스터디 모임에서 올린 문제를 풀이하고 다음 사람이 문제를 게시하고 스터디를 종료한다.
